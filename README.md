# Калькулятор сборов АВИА (servicefee)

Веб‑приложение на PHP + SQLite для расчёта **сервисного сбора**, **трансграна** и **эквайринга** по заказам (АВИА), с **историей** расчётов и **админкой правил**.

## Запуск

### Требования

- PHP 8.x

### Старт локально

Запуск встроенным сервером PHP из папки `servicefee/avia-calc/`:

```bash
php -S localhost:8888
```

Затем открыть в браузере:

- `http://localhost:8888/index.php` — калькулятор
- `http://localhost:8888/admin.php` — админка правил

## Возможности

- **Динамический пересчёт**: результат обновляется автоматически при вводе данных (с небольшой задержкой).
- **Валидация полей**: обязательные поля подсвечиваются **красной рамкой**, если без них нельзя выполнить расчёт.
- **Формулы**: в блоке «Применённые формулы» выводятся формулы в формате:  
  `X% от база = результат` (аналогично эквайрингу).  
  - Сервисный сбор — по услугам + итоговая строка.
  - Трансгран — одной строкой по заказу, от базы: **Сумма услуг + Сервисный сбор**.
- **История**: список последних расчётов, удаление записи/очистка истории.
- **Копирование**: кнопка «Скопировать» копирует в буфер текст **в формате блока «Применённые формулы»** (используется и на главной, и в истории).

## Структура

Основные файлы (см. также `servicefee/structure.md`):

- `servicefee/avia-calc/index.php` — UI калькулятора (вкладки Калькулятор/История)
- `servicefee/avia-calc/assets/app.js` — клиентская логика: сбор данных, авто‑расчёт, история, копирование
- `servicefee/avia-calc/assets/style.css` — стили (в т.ч. стиль в духе Битрикс24)
- `servicefee/avia-calc/api.php` — AJAX API (данные справочников, расчёт, история)
- `servicefee/avia-calc/src/Calculator.php` — расчёт сборов и формирование формул
- `servicefee/avia-calc/admin.php`, `servicefee/avia-calc/assets/admin.js` — админка правил

## Примечания

- База данных SQLite хранится в `servicefee/avia-calc/database/avia_calc.sqlite` и создаётся автоматически при первом запуске (см. `config/database.php`).
